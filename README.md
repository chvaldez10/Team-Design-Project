## Getting Started 🌟

To install all packages, run the command `pip install -r requirements.txt` 💼

Create an .env file with the following config:

```
ROOT_FOLDER=
EXPORT_DRIVE=
EXPORT_PATH_REMOTE=
EXPORT_PATH_LOCAL=
TALC_USERNAME=
```
